# Champion des Maths

## Description
Programme de révision mathématique en console, niveau CM1.
Mini-jeux : addition, soustraction, multiplication, division,
tables de multiplication et nombre manquant.

## Prérequis
- Compilateur GCC
- Système Linux, Windows ou Mac

## Installation et utilisation
1. Télécharger le fichier src/champion_maths.c
2. Compiler : gcc src/champion_maths.c -o champion_maths
3. Lancer : ./champion_maths

## Fonctionnalités
- 6 mini-jeux mathématiques
- Système de points (10 / 5 / 1 selon le nombre d'essais)
- Sauvegarde automatique des scores dans scores.txt
